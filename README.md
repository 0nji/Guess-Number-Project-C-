# Guess-Number-Project-C-
Simple guess the number game for the university project

Controls via inputing option numbers

3 Difficulties

Easy 1-50
Medium 1-100
Hard 1-250

Functions:
Main menu
High Scores TOP 5
Random respones
Save and read from save file scores.txt
Attempt counter
Return to menu after game completion

Launching:

```bash
g++ main.cpp -o guess
./guess
